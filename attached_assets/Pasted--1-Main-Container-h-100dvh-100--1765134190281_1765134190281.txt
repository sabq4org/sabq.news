تفاصيل تكوين الصفحة الرئيسية
1. الحاوية الرئيسية (Main Container)
- الارتفاع: h-[100dvh] (100% من ارتفاع الشاشة الديناميكي)
- العرض: w-full (عرض كامل)
- التمرير: overflow-y-scroll مع snap-y snap-mandatory (تمرير عمودي مع التقاط)
- الخلفية: bg-black (أسود)
- التمرير السلس: scroll-smooth
- إخفاء شريط التمرير: no-scrollbar

2. شعار UP news
- الموقع: fixed top-4 right-4 (ثابت، 16px من الأعلى واليمين)
- الطبقة: z-50 (أعلى طبقة)
- الخط: font-black text-xl tracking-tight (عريض جداً، حجم 20px)
- اللون: text-white (أبيض)
- يختفي عند فتح تفاصيل الخبر

3. بطاقة الخبر (NewsCard)
الحاوية:
- relative w-full h-full snap-start overflow-hidden bg-black text-white shrink-0
- (كامل العرض والارتفاع، نقطة التقاط، إخفاء الفائض)
الصورة الخلفية:
- absolute inset-0 (تغطي كامل البطاقة)
- object-cover opacity-80 (تغطية كاملة مع شفافية 80%)
- objectPosition: يعتمد على imageFocusX% و imageFocusY%
التدرج السينمائي:
- bg-gradient-to-t from-black via-black/60 to-transparent opacity-90
- (تدرج من أسود في الأسفل إلى شفاف في الأعلى)

4. حاوية المحتوى
- الموقع: absolute inset-x-0 bottom-0 (أسفل البطاقة)
- التباعد: p-6 pb-12 (padding: 24px، أسفل: 48px)
- التخطيط: flex flex-col items-start gap-4 (عمودي، المسافة بين العناصر: 16px)
- العرض الأقصى: max-w-md mx-auto w-full (448px في المنتصف)

5. شارة التصنيف (Metadata Badge)
التأثير الحركي (Framer Motion):
- initial: { opacity: 0, y: 20 } (يبدأ مخفي ومنزاح 20px للأسفل)
- whileInView: { opacity: 1, y: 0 } (يظهر في مكانه)
- transition: { duration: 0.5, delay: 0.1 } (مدة 0.5 ثانية، تأخير 0.1)
التنسيق:
- flex items-center gap-2 (أفقي، المسافة: 8px)
- text-xs font-medium text-white/70 (حجم صغير، شفافية 70%)
شارة التصنيف:
- bg-white/10 backdrop-blur-md px-3 py-1 rounded-full border border-white/10
- (خلفية شفافة 10%، ضبابية، حواف دائرية كاملة)

6. العنوان الرئيسي (Headline)
التأثير الحركي:
- initial: { opacity: 0, y: 20 }
- whileInView: { opacity: 1, y: 0 }
- transition: { duration: 0.5, delay: 0.2 } (تأخير 0.2 ثانية)
التنسيق:
- text-4xl (حجم 36px)
- font-bold (وزن عريض 700)
- leading-tight (ارتفاع السطر: 1.25)
- tracking-tight (تباعد الحروف: -0.025em)
- text-right w-full (محاذاة يمين، عرض كامل)
- font-sans (خط IBM Plex Sans Arabic)

7. الملخص (Summary)
التأثير الحركي:
- initial: { opacity: 0, y: 20 }
- whileInView: { opacity: 1, y: 0 }
- transition: { duration: 0.5, delay: 0.3 } (تأخير 0.3 ثانية)
التنسيق:
- text-lg (حجم 18px)
- text-gray-300 (رمادي فاتح)
- leading-relaxed (ارتفاع السطر: 1.625)
- max-w-[90%] (عرض أقصى 90%)
- text-right w-full (محاذاة يمين)

8. منطقة الأزرار (Actions)
التأثير الحركي:
- transition: { duration: 0.5, delay: 0.4 } (تأخير 0.4 ثانية)
التخطيط:
- w-full flex items-center justify-between mt-4 (عرض كامل، توزيع متساوي، هامش علوي 16px)
زر "اقرأ التفاصيل":
- text-black bg-white hover:bg-white/90 (نص أسود، خلفية بيضاء)
- border-none rounded-full px-6 font-bold (بدون حدود، دائري، padding: 24px)
مؤشر السحب:
- flex flex-col items-center gap-1 animate-bounce text-white/50
- السهم: w-6 h-6 (24px × 24px)
- النص: text-[10px] uppercase tracking-widest opacity-60

9. نافذة التفاصيل (NewsDetail Drawer)
الحاوية:
- h-[92vh] (92% من ارتفاع الشاشة)
- bg-black border-t border-white/10 text-white rounded-t-3xl
- (خلفية سوداء، حد علوي، زوايا دائرية 24px)
زر الإغلاق (السهم للأسفل):
- absolute top-6 left-1/2 -translate-x-1/2 z-50 (وسط أعلى، 24px من الأعلى)
- rounded-full bg-white/20 backdrop-blur-md w-10 h-10 (دائري 40px × 40px)
- أيقونة السهم: w-7 h-7 (28px × 28px)
المحتوى:
- pt-14 (padding علوي: 56px لإظهار زر الإغلاق)
- pb-20 (padding سفلي: 80px لمنطقة الأزرار)
الصورة: h-[40vh] (40% من ارتفاع الشاشة)
العنوان: text-3xl font-bold (30px، عريض)
النص: text-lg leading-relaxed text-gray-300 font-light (18px، خفيف)
ذيل الصفحة:
- bg-black/80 backdrop-blur-xl border-t border-white/10 p-4
- (شفافية 80%، ضبابية قوية)
