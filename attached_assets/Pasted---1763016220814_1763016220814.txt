# برومبت تنفيذي شامل - تطوير منصة سبق الذكية

## نظرة عامة على المشروع

### الهدف الرئيسي
تحويل منصة سبق الذكية إلى منصة إخبارية متقدمة مدعومة بالذكاء الاصطناعي مع نظام ملفات شخصية متكامل، وتجربة مستخدم استثنائية، وأداء تقني متقدم.

### نطاق العمل
- تحسين UI/UX الحالي
- تطوير نظام ملفات شخصية متقدم
- تعزيز الذكاء الاصطناعي والتخصيص
- تحسين الأداء والوصولية

---

## المرحلة الأولى: التحليل والتخطيط (أسبوعان)

### 1.1 تحليل الوضع الحالي
```
قم بإجراء تحليل شامل لمنصة سبق الذكية الحالية:
- تحليل بيانات المستخدمين وسجل التفاعل
- مراجعة الشكاوى والاقتراحات الحالية
- تحليل منافسيك (عربي بوست، مدى، الحرة)
- تحديد النقاط القوية والضعيفة
```

### 1.2 تحديد المتطلبات التفصيلية
```
ضع قائمة متطلبات دقيقة لكل عنصر:

أ. متطلبات الملف الشخصي:
- نظام تسجيل/دخول متقدم
- إدارة البيانات الشخصية
- تفضيلات المحتوى
- سجل القراءة والنشاط
- الشبكة الاجتماعية

ب. متطلبات الذكاء الاصطناعي:
- خوارزمية التوصية
- تحليل السلوك
- التخصيص الذكي
- التنبؤ بالاهتمامات

ج. متطلبات UI/UX:
- تحسين التصميم الحالي
- تعزيز الوصولية
- تحسين التجربة المتجاوبة
```

### 1.3 خطة المشروع التفصيلية
```
أنشئ خطة زمنية محددة:
- الجدول الزمني: 16 أسبوع
- الموارد المطلوبة: فريق تطوير (6 أشخاص)
- الميزانية التقديرية: 500,000 ريال سعودي
- المعايير القابلة للقياس: 85% رضا المستخدم
```

## المرحلة الثانية: تصميم وتطوير البنية التحتية (4 أسابيع)

### 2.1 تصميم قاعدة البيانات
```
صمم هيكل قاعدة بيانات متقدم:

1. جدول المستخدمين (users):
- id, email, password_hash, profile_data
- created_at, updated_at, last_login
- preferences, settings

2. جدول المقالات (articles):
- id, title, content, category, tags
- author_id, published_at, views_count
- sentiment_score, reading_time

3. جدول التفاعلات (interactions):
- id, user_id, article_id, interaction_type
- timestamp, duration, scroll_depth

4. جدول التفضيلات (preferences):
- id, user_id, category, weight
- last_updated, source_preferences

5. جدول الشبكة الاجتماعية (social_network):
- id, follower_id, following_id
- created_at, interaction_level
```

### 2.2 تطوير واجهات API
```
أنشئ واجهات برمجة التطبيقات:

1. Authentication API:
- POST /api/auth/register
- POST /api/auth/login
- POST /api/auth/logout
- POST /api/auth/refresh

2. User Profile API:
- GET /api/users/profile
- PUT /api/users/profile
- GET /api/users/{id}/activity
- GET /api/users/preferences

3. Content API:
- GET /api/articles/recommended
- GET /api/articles/trending
- POST /api/articles/{id}/save
- POST /api/articles/{id}/share

4. Social API:
- POST /api/social/follow
- POST /api/social/unfollow
- GET /api/social/followers
- GET /api/social/following
```

### 2.3 إعداد البنية التحتية السحابية
```
قم بإعداد البنية التحتية على AWS:

1. الخدمات المطلوبة:
- EC2 instances (2x t3.large)
- RDS PostgreSQL database
- ElastiCache Redis
- S3 for media storage
- CloudFront CDN

2. الأمان والمراقبة:
- AWS WAF for security
- CloudWatch for monitoring
- AWS Shield for DDoS protection
```

## المرحلة الثالثة: تطوير الواجهة الأمامية (4 أسابيع)

### 3.1 تحديث التصميم الحالي
```
طبق تحسينات UI/UX:

1. تحسين نظام الألوان:
- زيادة التباين بنسبة 20%
- إضافة وضع داكن/فاتح
- تحسين الألوان للوصولية

2. تحسين التوزيع:
- تقليل التكدس بنسبة 30%
- تحسين المساحات البيضاء
- إعادة ترتيب العناصر

3. تعزيز التفاعلات:
- إضافة رسوم متحركة سلسة
- تحسين زمن الاستجابة
- إضافة تغذية راجعة بصرية
```

### 3.2 تطوير صفحة الملف الشخصي
```
أنشئ واجهة ملف شخصية متقدمة:

1. الهيكل الرئيسي:
- Header مع صورة الغلاف والمعلومات الأساسية
- إحصائيات النشاط (مقالات مقروءة، محفوظة، متابعين)
- تبويبات للتنقل (القراءة، المحفوظات، التفضيلات)

2. المكونات التفاعلية:
- محرر الملف الشخصي
- إدارة التفضيلات
- سجل النشاط التفصيلي
- إدارة الشبكة الاجتماعية

3. الميزات المتقدمة:
- تخصيص الواجهة
- إعدادات الخصوصية
- إدارة الإشعارات
```

### 3.3 تطوير الميزات التفاعلية
```
أضف ميزات تفاعلية متقدمة:

1. نظام التعليقات:
- تعليقات متداخلة
- ردود معجبة/غير معجبة
- الإبلاغ عن التعليقات

2. مشاركة المحتوى:
- مشاركة على وسائل التواصل
- نسخ الرابط
- حفظ كملف PDF

3. المفضلة والمحفوظات:
- أنظمة مجلدات متعددة
- وسم المقالات
- البحث داخل المحفوظات
```

## المرحلة الرابعة: تكامل الذكاء الاصطناعي (3 أسابيع)

### 4.1 تطوير خوارزمية التوصية
```
أنشئ نظام توصية متعدد الطبقات:

1. التصفية التعاونية (Collaborative Filtering):
- تحليل سلوك المستخدمين المشابهين
- توصيات قائمة على التشابه
- معالجة البيانات الناقصة

2. التصفية المستندة إلى المحتوى:
- تحليل كلمات المفاتيح
- تصنيف المقالات
- التشابه الدلالي

3. التعلم العميق:
- نماذج RNN للتسلسل الزمني
- Attention mechanisms
- Transfer learning
```

### 4.2 تحليل السلوك والتنبؤ
```
نفذ نظام تحليل متقدم:

1. مقاييس التفاعل:
- وقت القراءة
- عمق التمرير
- معدل النقر
- نمط التصفح

2. التنبؤ بالسلوك:
- احتمالية القراءة
- وقت القراءة المتوقع
- الاهتمامات المستقبلية
- نسبة الارتداد
```

### 4.3 التخصيص الذكي
```
أضف ميزات تخصيص متقدمة:

1. الواجهة القابلة للتخصيص:
- ترتيب العناصر
- اختيار الأقسام
- حجم الخط والألوان
- تخطيط الصفحة

2. المحتوى المخصص:
- ترتيب المقالات
- اختيار الكُتّاب
- تفضيل المصادر
- مرشحات المحتوى
```

```

1. تحسين السرعة:

2. تحسين الخادم:

```

```
1. WCAG 2.1 AA Compliance:
- نسبة تباين 4.5:1 على الأقل
- دعم لوحة المفاتيح الكامل
- نصوص بديلة للصور
- تسميات واضحة للنماذج

2. دعم تقنيات المساعدة:
- ARIA labels
- Screen reader compatibility
- Voice navigation support
- High contrast mode
```

## الاستنتاج

هذا البرومبت يوفر خطة متكاملة لتطوير منصة سبق الذكية إلى منصة إخبارية متقدمة مدعومة بالذكاء الاصطناعي. النجاح يتطلب:

1. **التخطيط الدقيق:** تحليل شامل وتحديد واضح للأهداف
2. **التنفيذ المنهجي:** اتباع الخطوات بالترتيب مع التركيز على الجودة
3. **المراقبة المستمرة:** قياس الأداء وتعديل الخطة حسب الحاجة
4. **التحسين المستمر:** الاستجابة لتغذية المستخدمين وتطور السوق

النتيجة ستكون منصة إخبارية رائدة في المنطقة العربية، توفر تجربة مستخدم استثنائية وتعزز من مكانة سبق الذكية كرائدة في الإعلام الرقمي.
